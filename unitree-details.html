ROS-based Robot Guiding Dog System for Blind People

  1. Home
  2. Robot Guiding Dog System for Blind People

Key aspects

  • Simulation of the robot performing SLAM navigation algorithm (including path planning & obstacle avoidance) using Gazebo simulator.
  • Designing a haptic feedback unit to provide directional input to the user.
  • Comparing and assessing different feedback mechanisms to be used in the haptic feedback unit.
  • The feedback cues are provided to the user based on the measured distance from the robot.
  • The feedback unit uses an IR camera to locate an IR bar mounted on the back of the robot.
  • 3D printing of the various parts of the feedback unit, assembel and test the final product.

Project information

  • Type: Individual Project
  • Year: Third Year Undergraduate
  • Project date: 05 October, 2022
  • Project report: Report

Project details

This project revolved around developing a robotic guiding dog system to help blind people navigate their surrounding. The Robot ran the Robot Operating System (ROS) to control and manage the different part of the system and a navigation algorithm (SLAM) was implemented and simulated using Gazebo simulator as shown below.
A haptic feedback unit was designed to interact with the robot dog as wel as providing directional input to the user.
