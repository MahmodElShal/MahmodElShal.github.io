Project details

This project aims to develop a ROS-based mobile robot that can be fitted with various sensors such as a depth camera and a lidar unit allowing the robot to identify a target object and autonomously navigate its surroundings. A robotic arm will be attached to the robot to handle and retrieve the target object and return it to a goal position.

  • Assemble the base platform and add additional components before testing its functionality.
  • Design modular load sleds to fit the different components on the robot such as robotic manipulator, lidar unit, intel nuc (onboard PC), and depth camera.
  • Program the robot using ROS 2 so it can be controlled autonomously (main controller) or using a joystick (as a backup).
  • Train a computer vision ML model to detect the target objects and implement a navigation algorithm on the robot.
