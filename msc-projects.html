Simulation of a robot vacuum cleaner
ROS2 & Gazebo

  • Designed the URDF (Unified Robot Description Format) of the robot vacuum cleaner including its Lidar, Depth-camera, and IMU.
  • Implemented a differential drive controller for the robot.
  • Simulated the robot in Gazebo Simulator, implementing autonomous navigation, teleoperation, and behavior trees for functionality testing and optimization.

A* Path-Planning Algorithm

  • Developed a Python-based A* algorithm, integrated with a graphical user interface (GUI) to provide intuitive visualisation of the generated path.

2D Drone Control

  • Implemented and tuned a cascaded PID controller to control the position of the 2D drone based on the current position, and velocities feedback.
